## Featured projects

### 💳 [Visa Authorization Performance Dashboard](https://github.com/Mykeil-tzul/visa-auth-performance-dashboard)
Analyzes approval rates, decline drivers, and merchant segmentation using Python, 
scikit-learn, SQL, and Plotly. Built from my daily work at Visa.

### 🏀 [NBA Fatigue & Longevity Model](https://github.com/Mykeil-tzul/nba-fatigue-longevity-model)
Predicts performance dips and career longevity risk across 200+ players using 
minutes load, travel density, and back-to-back frequency. I played professionally — 
this one is personal.

### 🌆 [Atlanta Economic Mobility Analysis](https://github.com/Mykeil-tzul/atlanta-economic-mobility-analysis)
End-to-end clustering pipeline segmenting Atlanta zip codes by opportunity profile 
using Census API, Opportunity Atlas, and Folium.

### 🇧🇿 [Belize Climate Vulnerability Index](https://github.com/Mykeil-tzul/belize-climate-vulnerability-index)
ML model scoring Belizean districts by economic and climate risk. Random Forest + 
NLP sentiment analysis on local news. World Bank + NASA POWER data. My mother 
is from Belize — this one matters.

### ⚾ [MLB Pitch Outcome Predictor](https://github.com/Mykeil-tzul/mlb-pitch-outcome-predictor)
XGBoost classifier trained on 1.4M+ Statcast pitches predicting swing probability, 
contact rate, and strikeout likelihood. Deployed as a live Streamlit app.

### 🏠 [Family Budget Intelligence](https://github.com/Mykeil-tzul/family-budget-intelligence)
Personal finance pipeline: bank CSV exports → spend categorization → monthly 
drift and savings runway dashboard. Built because I needed it.
